# CampusBites Broken Demo

This is the intentionally weak version of CampusBites used by BuildProof to show common hackathon submission mistakes.

## Live links

- Vercel: https://buildproof-demo-bad.vercel.app
- GitHub: https://github.com/crvenimarlboro/BuildProof-demo-bad

## What this demo shows

- Homepage loads
- `/demo` does not exist
- `/docs` does not exist
- No working health endpoint

That makes it useful as a "broken submission" demo target for BuildProof.

## Local preview

Open `index.html` or serve this folder with any static file server.

## Deploy to Vercel

Deploy this directory as its own project. No build step is required.
